# DofusTabs

Herramienta para manejar múltiples ventanas de Dofus Retro de forma rápida y sin retrasos.

## 🚀 Características

- Elimina el delay del mouse al usar Alt+Escape o cambiar ventanas
- Cambio rápido entre ventanas con F1 / F2
- AutoFocus en eventos:
  - Turnos
  - Intercambios
  - Invitaciones a grupo
- Organización tipo tabs
- Loadouts (Layouts): guarda y aplica tu orden de ventanas fácilmente
- Ocultar/mostrar menú con F3

## 🛠 Requisitos

- Windows
- .NET (según versión del proyecto)
- **Npcap**

Descarga:
https://nmap.org/npcap/

> ⚠️ Importante: instalar Npcap en modo compatible con WinPcap

## ⚙️ Uso

- Ejecuta el programa
- Abre tus clientes de Dofus
- Usa F1 / F2 para moverte entre ventanas
- F3 para ocultar o mostrar el menú
- Configura layouts para ordenar tus cuentas
- El AutoFocus actuará automáticamente en eventos

## 📌 Nota

Proyecto de uso personal / educacional.  
Puede fallar y su uso podría ir en contra de los TOS de Ankama.

## ⚠️ Importante

Para que el AutoFocus funcione correctamente debes conectarte al personaje desde la selección.  
Si ya estás dentro del juego, reconecta.

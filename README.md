⚙️ Dofus Retro WindowTabs + AutoFocus

Herramienta personal para manejar múltiples ventanas de Dofus RETRO.

- Elimina el retraso de activación del mouse al usar Alt+Escape o cambiar ventanas
- AutoFocus en eventos (turno, trade, grupo)
- Organización tipo tabs
- Cambio rápido entre ventanas con F1 / F2
- Loadouts, Guarda tu orden favorito de personajes sin perder tiempo ordenando las Tabs. (En el programa dice Layouts)

📌 Nota
Proyecto solo para uso personal (backup/historial).
Con fines educacionales.
Puede fallar y su uso podría ir en contra de los TOS de Ankama.

⚠️ Importante
Para que el AutoFocus funcione correctamente debes conectarte al personaje 
(desde la selección de personaje). Si ya estás conectado, reconecta.

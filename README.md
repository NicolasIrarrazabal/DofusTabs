# DofusTabs

DofusTabs es una herramienta diseñada para facilitar la gestión de múltiples cuentas/ventanas de Dofus, optimizando el cambio entre instancias y eliminando el retraso típico de métodos tradicionales como Alt+Escape o herramientas similares (como WindowTabs).

## 🚀 Características

- ⚡ **Cambio de ventanas optimizado**
  - Eliminación del delay de mouse producido por Alt+Escape
  - Respuesta inmediata al cambiar entre cuentas

- 🎯 **Auto-enfoque inteligente**
  - Detecta eventos en tiempo real:
    - Turnos
    - Intercambios
    - Invitaciones a grupo
  - Enfoca automáticamente la ventana correspondiente

- 📂 **Loadouts de pestañas**
  - Guarda configuraciones de ventanas
  - Permite ordenar rápidamente tus cuentas

- ⌨️ **Atajos de teclado**
  - `F1`: cambiar a la pestaña anterior
  - `F2`: cambiar a la pestaña siguiente
  - `F3`: ocultar/mostrar el menú

## 🛠 Requisitos

- Windows
- .NET (según versión del proyecto)
- **Npcap**

Descarga:
https://nmap.org/npcap/
